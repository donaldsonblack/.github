## 🌐 What We Do
- 🖥️ Modern web applications with **React**, **TypeScript**, and **TailwindCSS**  
- ⚙️ Resilient backends with **Java + Spring Boot**

- 🚀 Automation and CI/CD pipelines with **Gradle** and GitHub Actions
